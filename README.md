## Sobre

Este repositório reúne conteúdos desenvolvidos ao longo de uma formação em desenvolvimento Full Stack, organizados por módulos progressivos ("Mundos"), cobrindo desde lógica de programação até aplicações completas envolvendo frontend, backend, cloud e mobile.

##

## Estrutura do curso

* [Mundo 01](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo01) - Programador Raiz em Python - Lógica de programação com Python e resolução de problemas (BeeCrowd).

* [Mundo 02](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02) - Programador Front-End - Desenvolvimento Frontend com HTML, CSS e JavaScript, incluindo frameworks modernos.

* [Mundo 03](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo03) - Programador Back-End - Desenvolvimento Backend com Java e banco de dados SQL.

* [Mundo 04](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo04) - Programador Cloud & Mobile - Aplicações Cloud e Mobile com tecnologias como Azure, Flutter e Android.

* [Mundo 05](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo05) - Gestão & Negócios - Gestão, dados e fundamentos de negócios, incluindo metodologias ágeis e análise de dados.
