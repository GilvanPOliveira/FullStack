# Angular

Aplicação frontend desenvolvida com Angular utilizando componentes, serviços e roteamento.

##

## Sobre

Projeto focado na construção de uma aplicação web utilizando Angular, explorando conceitos fundamentais como arquitetura baseada em componentes, injeção de dependência, formulários e navegação entre páginas.

Este projeto simula um sistema de gerenciamento de livros, com operações de listagem, inclusão e remoção de dados.

##

## Funcionalidades

* listagem de itens em tabela
* inclusão de novos registros via formulário
* remoção de registros
* navegação entre páginas (lista e cadastro)
* uso de serviços para controle de dados
