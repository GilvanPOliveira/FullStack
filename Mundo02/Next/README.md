# Next.js

Aplicação web desenvolvida com Next.js e TypeScript para gerenciamento de livros.

##

## Sobre

Projeto focado na construção de uma aplicação frontend com Next.js, explorando componentes reutilizáveis, tipagem com TypeScript e consumo de dados por meio de API interna.

O sistema simula um catálogo de livros com operações de listagem, inclusão e exclusão, utilizando rotas internas para gerenciamento dos dados.

##

## Funcionalidades

* listagem de livros
* inclusão de novos registros
* exclusão de itens
* navegação entre páginas
* uso de API interna com rotas Next.js
