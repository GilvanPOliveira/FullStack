# Vue

Aplicação frontend desenvolvida com Vue.js para manipulação de dados e construção dinâmica de interfaces.

##

## Sobre

Projeto focado na construção de aplicações web utilizando JavaScript e Vue.js, explorando manipulação de DOM, consumo de dados em JSON e renderização dinâmica de conteúdo.

O sistema apresenta diferentes cenários de uso, incluindo ordenação de dados, exibição de catálogo e integração com APIs externas.

##

## Funcionalidades

* manipulação de dados com JavaScript
* ordenação e transformação de listas
* geração dinâmica de elementos HTML
* consumo de API REST
* renderização de dados com Vue.js
