# Projeto Integrador - OneQuiz

Aplicação web de quiz interativo com foco em experiência do usuário.

##

## Sobre

Projeto desenvolvido como aplicação frontend interativa, permitindo que usuários respondam perguntas, acompanhem pontuação e naveguem entre questões de forma dinâmica.

##

## Funcionalidades

* exibição de perguntas e alternativas
* seleção e validação de respostas
* contagem de pontuação
* navegação entre perguntas
* exibição de resultado final
