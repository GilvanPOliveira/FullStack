## Mundo 02

Módulo focado em desenvolvimento Frontend utilizando múltiplas tecnologias e frameworks.

##

## Sobre

Este diretório reúne projetos desenvolvidos com foco em aplicações frontend, abordando desde HTML e CSS até frameworks modernos e integração com backend.

##

## Estrutura do Mundo 02

[Nível 01](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02/Html) - HTML e CSS — estruturação de páginas e fundamentos de layout. 

[Nível 02](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02/Vue) - JavaScript e Vue — manipulação de DOM, JSON e construção dinâmica de interfaces. 

[Nível 03 - (parte 01)](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02/React) - React — construção de aplicações com componentes reutilizáveis. 

[Nível 03 - (parte 02)](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02/Next) - Next.js — frontend com API interna e renderização otimizada. 

[Nível 04](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02/Angular) - Angular — arquitetura baseada em serviços, componentes e rotas. 

[Nível 05](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02/MongoDb) - Integração com backend — Node.js, Express e MongoDB. 

[Projeto Integrador](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo02/OneQuiz) - Aplicação de quiz interativo com foco em experiência do usuário.
