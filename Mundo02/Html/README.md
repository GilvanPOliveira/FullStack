# HTML

Aplicação web estática desenvolvida com HTML e CSS para apresentação de receitas.

##

## Sobre

Projeto focado na construção de um site estático utilizando HTML e CSS, explorando estrutura semântica, organização de layout e estilização de páginas.

O sistema simula um site de receitas com navegação entre páginas e formulário de cadastro.

##

## Funcionalidades

* página inicial com apresentação do site
* listagem de receitas com layout em cards
* formulário de cadastro com validação
* navegação entre páginas
* uso de Bootstrap para layout e responsividade
