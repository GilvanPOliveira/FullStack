# MongoDB

Aplicação com backend em Node.js utilizando Express e MongoDB, integrada a múltiplos frontends.

##

## Sobre

Projeto focado na construção de uma aplicação completa com backend e integração frontend, utilizando MongoDB como banco de dados.

O sistema implementa uma API para gerenciamento de dados e conecta diferentes aplicações frontend, demonstrando comunicação entre cliente e servidor.

##

## Funcionalidades

* API REST para gerenciamento de dados
* operações de listagem, criação e exclusão
* integração com banco de dados MongoDB
* comunicação entre backend e múltiplos frontends
* suporte a aplicações em React, Next.js e Angular
