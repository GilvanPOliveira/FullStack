# React

Aplicação frontend desenvolvida com React e TypeScript para gerenciamento de livros.

##

## Sobre

Projeto focado na construção de uma aplicação web utilizando React, explorando componentes reutilizáveis, gerenciamento de estado e navegação entre páginas.

O sistema simula um catálogo de livros com funcionalidades de listagem, inclusão e exclusão de registros.

##

## Funcionalidades

* listagem de livros em tabela
* inclusão de novos registros via formulário
* exclusão de itens
* navegação entre páginas
* gerenciamento de estado com hooks
