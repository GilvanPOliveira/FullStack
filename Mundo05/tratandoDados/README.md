# Nível 03 — Tratamento de Dados com Python e Pandas

Exercício focado na **limpeza e preparação de dados** para utilização em processos de análise e mineração de dados.

##

## Sobre

Esta atividade aborda o processo de **data cleaning**, uma etapa essencial em projetos de análise de dados e ciência de dados.

O exercício simula o trabalho de um **Analista de Dados**, responsável por preparar um conjunto de dados antes que ele seja utilizado em processos de análise ou descoberta de conhecimento.

Durante a atividade são aplicadas técnicas de tratamento de dados utilizando **Python e a biblioteca Pandas**, incluindo substituição de valores nulos, transformação de tipos de dados e remoção de registros inconsistentes.

##

## Objetivo

Aplicar técnicas de preparação de dados para tornar um dataset adequado para análise, incluindo:

* leitura de dados a partir de arquivos CSV
* identificação e tratamento de valores nulos
* transformação de tipos de dados
* padronização de formatos
* remoção de registros inconsistentes

##

## Conceitos Aplicados

* limpeza de dados (data cleaning)
* preparação de datasets
* manipulação de dados com Pandas
* transformação de dados

##

## Ferramentas e Tecnologias

* Python
* Pandas
* Jupyter Notebook / Scripts Python
* Manipulação de arquivos CSV
