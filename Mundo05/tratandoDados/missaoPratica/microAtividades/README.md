# Missão Prática — Tratamento de Dados com Pandas

Exercício prático focado na **limpeza e preparação de dados** utilizando Python e a biblioteca Pandas.

## 

## Sobre

Esta atividade aborda o processo de **tratamento e preparação de dados (data cleaning)**, etapa essencial em projetos de análise de dados e ciência de dados.

O exercício simula o trabalho de um **Analista de Dados**, responsável por preparar um conjunto de dados para que ele possa ser utilizado em processos de mineração de dados e descoberta de conhecimento.

Durante a atividade são aplicadas técnicas de manipulação e limpeza de dados utilizando **Python e a biblioteca Pandas**, incluindo tratamento de valores nulos, padronização de formatos e remoção de registros inconsistentes.

##

## Objetivo

Aplicar técnicas de preparação e limpeza de dados para tornar um dataset adequado para análise, incluindo:

* leitura de dados a partir de arquivos CSV
* identificação e tratamento de valores nulos
* padronização de formatos de data
* transformação de tipos de dados
* remoção de registros inconsistentes

##

## Conceitos Aplicados

* limpeza de dados (data cleaning)
* manipulação de datasets com Pandas
* transformação de tipos de dados
* preparação de dados para análise

##

## Ferramentas e Tecnologias

* Python  
* Pandas  
* Manipulação de arquivos CSV  
* Scripts Python / Jupyter Notebook
