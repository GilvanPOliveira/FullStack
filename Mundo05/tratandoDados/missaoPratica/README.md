# Missão Prática — Tratamento de Dados com Python e Pandas

Exercício prático focado na **limpeza e preparação de dados** para utilização em tarefas de análise e mineração de dados.

##

## Sobre

Esta missão prática aborda o processo de **tratamento de dados (data cleaning)**, etapa fundamental em projetos de análise de dados e ciência de dados.

Durante a atividade, o aluno assume o papel de **Analista de Dados**, responsável por preparar um conjunto de dados para que ele possa ser utilizado em processos de descoberta de conhecimento.

O exercício utiliza **Python e a biblioteca Pandas** para realizar operações de leitura, transformação e limpeza de dados provenientes de um arquivo CSV.

##

## Objetivo

Aplicar técnicas de tratamento e preparação de dados para tornar um dataset adequado para análise, incluindo:

* leitura de dados a partir de arquivos CSV
* identificação e tratamento de valores nulos
* padronização de formatos de data
* transformação de tipos de dados
* remoção de registros inconsistentes

##

## Conceitos Aplicados

* limpeza de dados (data cleaning)
* manipulação de datasets com Pandas
* transformação de tipos de dados
* preparação de dados para análise

##

## Ferramentas e Tecnologias

* Python  
* Pandas  
* Manipulação de arquivos CSV  
* Scripts Python / Jupyter Notebook
