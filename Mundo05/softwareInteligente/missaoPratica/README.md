# Missão Prática — Análise de Sentimento com PLN

Exercício prático focado na aplicação de **Processamento de Linguagem Natural (PLN)** para análise de sentimento em textos.

##

## Sobre

Esta atividade simula um cenário real de análise de dados, onde uma empresa busca entender a percepção do público em relação a clubes de futebol a partir de conteúdos publicados em redes sociais.

O projeto utiliza técnicas de PLN para analisar sentimentos expressos em textos (tweets), permitindo identificar tendências positivas, negativas ou neutras.

A implementação é realizada em ambiente de nuvem (Google Colab), utilizando bibliotecas modernas de processamento de linguagem natural.

##

## Objetivo

Aplicar técnicas de análise de sentimento em textos, incluindo:

* processamento de linguagem natural (PLN)
* análise de polaridade de textos
* tratamento e interpretação de dados textuais
* execução de modelos em ambiente cloud

##

## Conceitos Aplicados

* Processamento de Linguagem Natural (PLN)
* análise de sentimento (sentiment analysis)
* polaridade de texto (-1 a 1)
* pipeline de NLP
* análise de dados não estruturados

##

## Ferramentas e Tecnologias

* Python
* Google Colab
* spaCy
* SpacyTextBlob
