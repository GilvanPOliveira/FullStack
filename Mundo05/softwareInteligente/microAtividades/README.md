# Microatividades — Processamento de Linguagem Natural (PLN)

Conjunto de exercícios práticos focados na aplicação de técnicas de **Processamento de Linguagem Natural** utilizando Python.

##

## Sobre

Estas atividades exploram conceitos fundamentais de análise de texto e inteligência aplicada, utilizando ferramentas modernas em ambiente de execução na nuvem.

O fluxo aborda desde a configuração do ambiente até a aplicação de técnicas de PLN, permitindo a extração de informações relevantes a partir de dados textuais.

As práticas são realizadas com suporte de bibliotecas como spaCy, PKE e langdetect, simulando cenários reais de análise de dados.

##

## Objetivo

Aplicar técnicas de processamento de linguagem natural, incluindo:

* configuração de ambiente em nuvem (Google Colab)
* análise e processamento de textos
* extração de informações relevantes
* identificação de padrões linguísticos

##

## Conceitos Aplicados

* Processamento de Linguagem Natural (PLN)
* tokenização e análise linguística
* reconhecimento de entidades (NER)
* extração de palavras-chave
* detecção de idioma

##

## Ferramentas e Tecnologias

* Python
* Google Colab
* spaCy
* PKE (Keyphrase Extraction)
* langdetect
