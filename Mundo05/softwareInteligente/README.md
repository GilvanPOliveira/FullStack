# Análise de Sentimentos com PLN

Projeto de análise de sentimentos utilizando técnicas de Processamento de Linguagem Natural (PLN) aplicadas a textos de redes sociais.

##

## Sobre

Este projeto demonstra a aplicação de técnicas de **Processamento de Linguagem Natural (PLN)** para identificar sentimentos em textos publicados em redes sociais.

A análise é realizada a partir de tweets relacionados a clubes de futebol, com o objetivo de identificar percepções positivas ou negativas expressas pelos usuários.

A solução utiliza bibliotecas de NLP em Python para processar os textos e gerar um **score de polaridade**, indicando o sentimento associado a cada mensagem.

##

## Objetivo

Aplicar técnicas de **análise de sentimentos** em textos utilizando ferramentas de processamento de linguagem natural.

O projeto envolve:

* coleta e definição de textos para análise
* processamento linguístico
* identificação de polaridade de sentimentos
* classificação de mensagens como positivas ou negativas

##

## Processamento de Linguagem Natural

A análise de sentimentos utiliza um modelo linguístico capaz de avaliar o contexto de uma frase e atribuir um valor numérico de polaridade.

A escala utilizada varia entre:

* **-1** → sentimento negativo
* **0** → neutro
* **1** → sentimento positivo

Cada texto analisado recebe um score indicando a percepção geral do conteúdo.
