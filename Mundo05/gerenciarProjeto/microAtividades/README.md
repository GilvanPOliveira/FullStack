# Microatividades — Gestão de Projetos

Conjunto de exercícios práticos voltados ao aprendizado de fundamentos de gestão de projetos.

##

## Sobre

Este diretório reúne microatividades desenvolvidas para explorar conceitos fundamentais de planejamento e gestão de projetos.

As atividades abordam desde definição de objetivos até identificação de riscos, utilizando práticas comuns em metodologias de gerenciamento de projetos.

##

## Atividades

**Microatividade 01**
Definição de objetivos utilizando a metodologia **SMART**, estruturando metas específicas, mensuráveis, atingíveis, relevantes e temporais.

**Microatividade 02**
Criação de uma **Estrutura Analítica do Projeto (EAP)** para organização das fases e entregas de um projeto.

**Microatividade 03**
Definição e documentação de **requisitos de projeto**, detalhando necessidades e condições para execução das atividades.

**Microatividade 04**
Identificação e organização de **marcos do projeto**, representando eventos importantes no cronograma.

**Microatividade 05**
Identificação e análise de **riscos do projeto**, incluindo impacto, responsáveis e estratégias de mitigação.

##

## Ferramentas e Conceitos

* Planejamento de projetos
* Metodologia SMART
* Estrutura Analítica do Projeto (EAP)
* Gestão de riscos
* Documentação de requisitos
