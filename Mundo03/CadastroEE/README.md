# CadastroEE

Aplicação backend web em Java utilizando Servlets, JPA e Jakarta EE.

##

## Sobre

Projeto focado na construção de um sistema cadastral com interface web, integrando persistência de dados, controle de fluxo e renderização de páginas no servidor.

A aplicação utiliza Java Enterprise Edition para implementar cadastro, listagem, alteração e exclusão de registros, demonstrando integração entre camada web, camada de negócio e banco de dados.

##

## Funcionalidades

* listagem de registros em interface web
* inclusão de novos dados
* alteração de registros existentes
* exclusão de dados
* navegação controlada por servlet
* persistência com JPA

##

## Arquitetura

O projeto é dividido em camadas com responsabilidades bem definidas:

**Persistência**

* entidades JPA
* acesso ao banco de dados via pool de conexões

**Controle**

* Session Beans
* fachada para operações de negócio

**Web**

* Servlets para controle das requisições
* JSPs para renderização das páginas

## Execução

A aplicação é executada em ambiente servidor, com interface web para gerenciamento dos registros e integração com banco de dados relacional.
