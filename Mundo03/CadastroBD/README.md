# CadastroBD

Aplicação backend em Java com persistência de dados utilizando SQL Server e JDBC.

##

## Sobre

Projeto focado na construção de uma aplicação Java com acesso a banco de dados relacional utilizando JDBC.

O sistema implementa um modelo de cadastro de pessoas físicas e jurídicas, utilizando princípios de orientação a objetos, mapeamento objeto-relacional e padrão DAO para manipulação dos dados.

##

## Funcionalidades

* cadastro de pessoas físicas e jurídicas
* persistência de dados em banco SQL Server
* operações CRUD completas
* listagem de registros
* manipulação de dados via console

##

## Arquitetura

O projeto segue uma estrutura baseada em separação de responsabilidades:

**Model**

* classes de domínio (`Pessoa`, `PessoaFisica`, `PessoaJuridica`)

**DAO**

* camada de acesso a dados
* operações CRUD utilizando JDBC

**Util**

* gerenciamento de conexão com banco
* controle de sequências e consultas

**Application**

* classe principal responsável pela execução e testes do sistema

##

## Stack

[![My Skills](https://skillicons.dev/icons?i=java,mysql)](https://skillicons.dev)

* Java
* JDBC
* SQL Server
* DAO Pattern

##

## Execução

A aplicação é executada via console, permitindo realizar operações de cadastro, consulta, alteração e exclusão de registros diretamente no banco de dados.
