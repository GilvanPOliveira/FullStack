# Mundo 03

Módulo focado em desenvolvimento Backend utilizando Java e banco de dados SQL.

##

## Sobre

Este diretório reúne projetos desenvolvidos com foco em construção de aplicações backend, explorando conceitos como orientação a objetos, persistência de dados e integração entre sistemas.

Os projetos evoluem progressivamente desde aplicações Java básicas até sistemas completos com integração com banco de dados e execução em ambiente servidor.

##

## Estrutura do Mundo 03

[Nível 01](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo03/CadastroPoo) - Fundamentos de programação orientada a objetos com Java.

[Nível 02](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo03/Loja) - Persistência de dados utilizando banco de dados SQL. 

[Nível 03](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo03/CadastroBD) - Integração entre aplicação Java e banco de dados.

[Nível 04](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo03/CadastroEE) - Aplicação corporativa utilizando Java Enterprise Edition.

[Nível 05](https://github.com/GilvanPOliveira/FullStack/tree/main/Mundo03/CadastroServidor) - Sistema completo com integração entre backend e banco de dados.
