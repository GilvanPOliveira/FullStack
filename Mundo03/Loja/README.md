# Loja

Projeto de modelagem e implementação de banco de dados utilizando SQL Server.

##

## Sobre

Projeto focado na criação de um banco de dados para gerenciamento de produtos, usuários, clientes e movimentações de estoque.

O sistema modela operações de compra e venda, registrando movimentações realizadas por operadores do sistema e mantendo o controle de entradas e saídas de produtos.

##

## Funcionalidades

* cadastro de usuários operadores
* cadastro de pessoas físicas e jurídicas
* cadastro de produtos
* registro de movimentações de compra e venda
* consultas analíticas sobre operações realizadas

##

## Estrutura do Modelo

O banco de dados é composto por entidades principais relacionadas ao fluxo de movimentação de produtos:

**Usuários**

* operadores responsáveis pelas operações no sistema

**Pessoas**

* entidade base para identificação de clientes e fornecedores
* especialização em pessoa física e pessoa jurídica

**Produtos**

* cadastro de itens com controle de quantidade e preço

**Movimentações**

* registro de entradas (compras) e saídas (vendas)
* relacionamento entre produto, pessoa e operador

## Consultas Implementadas

O projeto inclui consultas para análise das operações do sistema, como:

* listagem de pessoas físicas e jurídicas
* movimentações de entrada e saída
* valor total de operações por produto
* valor total de operações por operador
* cálculo de média de venda por produto
