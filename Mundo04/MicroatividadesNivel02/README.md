# Microatividades — Flutter

Conjunto de exercícios práticos para exploração de widgets e construção de interfaces mobile utilizando Flutter.

##

## Sobre

Este diretório reúne atividades práticas voltadas ao aprendizado dos fundamentos do desenvolvimento mobile com Flutter.

Os exercícios exploram configuração do ambiente, estrutura básica de aplicações, organização de layouts e construção de interfaces interativas utilizando diferentes widgets.

##

## Atividades

**Microatividade 01**
Preparação do ambiente de desenvolvimento Flutter.

**Microatividade 02**
Construção da estrutura básica de um aplicativo utilizando `MaterialApp`, `Scaffold` e `AppBar`.

**Microatividade 03**
Criação de layouts utilizando widgets como `Column`, `Row`, `Text` e `Icon`.

**Microatividade 04**
Implementação de listas dinâmicas utilizando o widget `ListView`.

**Microatividade 05**
Customização de interfaces com widgets avançados como `Stack`.
