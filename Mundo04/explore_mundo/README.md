# Explore Mundo

Aplicação mobile desenvolvida com Flutter para apresentação de destinos turísticos e interação com usuários.

##

## Sobre

Projeto de aplicativo mobile criado para uma agência de viagens fictícia chamada **Explore Mundo**.

O objetivo da aplicação é permitir que usuários explorem destinos turísticos, consultem pacotes de viagem e visualizem avaliações de outros clientes, utilizando uma interface interativa construída com Flutter.

##

## Funcionalidades

* exibição de destinos turísticos em destaque
* navegação entre seções do aplicativo
* busca rápida por destinos ou pacotes
* visualização de avaliações de clientes
* interface construída com widgets Flutter
