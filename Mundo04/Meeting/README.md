# Meeting

Aplicação mobile desenvolvida com React Native para cadastro e gerenciamento de fornecedores.

##

## Sobre

Projeto de aplicativo mobile voltado para gerenciamento de fornecedores, permitindo cadastro, visualização e organização de informações relevantes como contato, endereço e categorias de produtos.

A aplicação foi desenvolvida utilizando React Native e Expo, com foco em experiência de usuário simples e eficiente.

##

## Funcionalidades

* cadastro de fornecedores
* listagem de fornecedores cadastrados
* associação de imagens aos perfis
* interface mobile intuitiva
* pesquisa e filtragem de fornecedores
