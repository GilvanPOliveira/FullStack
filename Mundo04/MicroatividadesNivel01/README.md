# Microatividades — React Native

Conjunto de exercícios práticos para exploração de componentes e funcionalidades básicas do React Native.

##

## Sobre

Este diretório reúne pequenas atividades práticas desenvolvidas para explorar os fundamentos do desenvolvimento mobile com React Native.

Os exercícios demonstram manipulação de entrada de texto, renderização dinâmica de listas e uso de imagens para construção de interfaces interativas.

##

## Atividades

**Microatividade 01**
Configuração do ambiente de desenvolvimento React Native.

**Microatividade 02**
Entrada de texto com `TextInput` e transformação dinâmica do conteúdo exibido.

**Microatividade 03**
Criação de listas dinâmicas utilizando `ScrollView`.

**Microatividade 04**
Implementação de visualização eficiente de listas com renderização otimizada.

**Microatividade 05**
Uso de imagens em interfaces mobile.
