# Doma

Aplicação wearable desenvolvida para dispositivos Wear OS com foco em acessibilidade e interação por áudio.

##

## Sobre

Projeto de aplicação para dispositivos Wear OS voltado à assistência de funcionários em ambiente corporativo.

O sistema utiliza recursos de áudio e integração com sensores do dispositivo para fornecer notificações, leitura de mensagens e alertas de segurança, auxiliando principalmente usuários com deficiência visual e melhorando a comunicação no ambiente de trabalho.

##

## Funcionalidades

* leitura de mensagens e notificações em voz alta
* respostas a comandos de voz
* alertas de segurança e notificações sonoras
* suporte a dispositivos Bluetooth para saída de áudio
* detecção dinâmica de dispositivos de áudio

## Stack

[![My Skills](https://skillicons.dev/icons?i=java,androidstudio)](https://skillicons.dev)

* Java
* Android Studio
* Wear OS
* APIs de áudio do Android
