# LogiMove

Projeto de banco de dados em nuvem utilizando Azure para gerenciamento de operações logísticas.

##

## Sobre

Projeto focado na construção de uma solução de banco de dados em nuvem para apoiar operações de uma empresa de logística.

A aplicação utiliza serviços do Microsoft Azure para armazenar e gerenciar dados relacionados a motoristas, clientes e pedidos, demonstrando o uso de infraestrutura cloud para suportar aplicações corporativas.

##

## Funcionalidades

* configuração de banco de dados em nuvem
* modelagem de dados para sistema logístico
* criação de tabelas e relacionamentos
* inserção e gerenciamento de dados
* execução de consultas analíticas
* operações CRUD completas

##

## Estrutura do Sistema

O banco de dados modela os principais elementos do fluxo logístico:

**Motoristas**

* cadastro de profissionais responsáveis pelo transporte

**Clientes**

* registro de empresas ou pessoas que solicitam serviços

**Pedidos**

* controle de solicitações de transporte e movimentação de cargas
