# IoT Azure — Telemetria em Tempo Real

Projeto de monitoramento de sensores utilizando Azure IoT Hub e visualização em tempo real via aplicação web.

##

## Sobre

Projeto voltado à coleta e visualização de dados de sensores em tempo real utilizando infraestrutura de Internet das Coisas (IoT) na nuvem.

O sistema simula dispositivos conectados enviando dados de temperatura e umidade para o **Azure IoT Hub**, permitindo que essas informações sejam processadas e exibidas em uma aplicação web.

##

## Funcionalidades

* coleta de dados de sensores simulados
* envio de telemetria para o Azure IoT Hub
* visualização em tempo real dos dados
* processamento de mensagens em formato JSON
* integração com aplicação web hospedada na nuvem

##

## Arquitetura

O fluxo de dados segue a seguinte estrutura:

**Dispositivo IoT**

* simulação de sensores (temperatura e umidade)

**Azure IoT Hub**

* recebimento e gerenciamento de telemetria

**Aplicação Web**

* exibição dos dados em tempo real

**Serviços Azure (opcional)**

* armazenamento em Blob Storage
* alertas automatizados
* integração com serviços adicionais
